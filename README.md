# WhatFlower

## 🎯 WhatFlower는?
**🗣️ CoreML과 Vision Framework, API를 이용한 꽃 분류 App 입니다.**

<br>

## 🚀 주요 기능
- 꽃을 감지하고 식별하며, 꽃에 대한 분류와 설명을 보여줍니다.

<br>

## 🛠️ 기술스택
- Swift (version 5.9.2)
- Xcode (version 15.1)
- UIKit Framework
- CoreML
- Vision Framework
- Wikipedia API
- [CocoaPods] Alamofire
- [CocoaPods] SwiftyJSON
- [CocoaPods] SDWebImage

<br>
